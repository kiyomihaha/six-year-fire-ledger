# 六年之约记账

一个手机优先的本地记账应用。核心思路是离线可用、操作轻、语气温和，把“六年之约”作为给未来自己的心理暗示。

## 功能

- 快速记录支出、收入、存入
- 总览今日日期、本月结余、今日花费、愿景进度
- 消费分析：今日、本月、今年、本月日均、主要分类
- 随机温馨建议彩蛋
- 本机离线保存
- 手动备份到电脑、从电脑恢复

## 本地运行

```bash
node server.js
```

电脑访问：

```text
http://127.0.0.1:4174
```

手机和电脑在同一个 Wi-Fi 下时，手机访问电脑局域网地址，例如：

```text
http://192.168.11.106:4174
```

## 数据说明

账本数据会保存在浏览器本地。电脑备份数据保存在 `ledger-data.json`，这个文件已加入 `.gitignore`，不会上传到 GitHub。
